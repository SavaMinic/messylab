Academic projects

(2010) Messy Lab

Developed an Assembly IDE with interpretation and debugging capabilities as a part of my master’s thesis. It is intended to aid teaching of assembly languages and it is written in C#.

(2009) Automobile Cruise Control System

A real-time system with a GUI simulator, written in Java Real-time and deployed on SUSE Linux Enterprise Server. I developed the system by writing the Java code directly and by using the generated code from the detailed Rational Rose RT model. The purpose of the project was to develop a fully functional real-time application using both, the traditional and the model-driven approach. The schedulability analysis was a part of the project.

(2008) Airport Baggage Transport System

Includes a custom hardware schematics based around the 8086 processor and the 8051 microcontroller, as well as simulators written in C and x86 ASM. Firmwares were implemented in 8086 ASM and Keil C51. My role was to develop the simulators and the firmware in x86 and to participate in the overall system design.

(2008) MicroJava to C# translator

MicroJava is a homebrew language that resembles Java, but is much simpler. In implementing the translator, I used jFlex for lexer, and CUP for parser generation. The translator is written in Java.

(2007) Distributed Image Processing

A project with emphasis on synchronization protocols between clients, servers and working nodes. I designed the system to be fault tolerant and implemented it in Java. It utilizes the Bag-of-Tasks technique to distribute jobs and sockets for communication. The project includes a performance analysis.

(2006) Preemptive Multithreading Kernel

Implemented the kernel in C++ and some assembly. It was designed to run under MS-DOS and includes synchronization capabilities using semaphores, event handling, I/O and storage devices.

(2006) Automated Testing and Reporting

Conducted Black-box testing by separating of Classes of equivalence using Rational Suite. Written in VBA-like macro. The tested application was WinRAR.
